Mis proyectos

Proyecto nº 1 : Peluquería Canina en Java

Proyecto desarrollado en Java, referente a una aplicación de peluquería canina realizado en el curso de Programacion Orientada a objetos de TodoCode. Este sistema está diseñado siguiendo una arquitectura en capas que abarca la interfaz gráfica de usuario, la lógica y la persistencia de datos. La aplicación mantiene una relación de uno a uno entre las mascotas y sus dueños. Características principales: Registro de mascotas Visualización de mascotas registradas en una pantalla independiente Edición de la información de las mascotas Eliminación de registros de mascotas

Proyecto nº 2 : Login con sistema de Roles

Este proyecto, realizado en el curso de Programación Orientada a Objetos en Java de TodoCode, es una aplicación de escritorio desarrollada en Java que implementa un sistema de login complementado con un sistema de roles y usuarios. Dependiendo del rol del usuario (administrador o usuario), se desplegarán diferentes ventanas con funcionalidades específicas.
